# A LowLevelFEM Julia csomag használata az oktatásban

Az egyes kidolgozott feladatok az `examples` könyvtárban találhatók

* A használathoz telepíteni kell a [julia](https://julialang.org/) (opcionálisan a  [gmsh](https://gmsh.info)) programokat. Utóbbiból az *SDK*-t.
* A `julia` programozási nyelvhez telepíteni kell a [LowLevelFEM](https://juliahub.com/ui/Packages/General/LowLevelFEM) csomagot.

Megköszönök minden észrevételt és hibajelentést.
